# Online Voting System (Frontend Simulation)

The Online Voting System is a professional frontend simulation mini project created using HTML, CSS, and Vanilla JavaScript to demonstrate the working process of a digital election platform. This project was developed to create awareness about modern online voting systems and to provide a practical visualization of secure web-based voting procedures.

The application begins with a voter verification form where users enter basic details such as voter name, voter ID, mobile number, and age. JavaScript-based form validation ensures that only eligible users can proceed to the voting dashboard. The system then displays a dynamic list of candidates where the voter can select only one candidate and submit their vote.

To simulate a real election process, the project implements one-vote-per-user logic using browser localStorage, preventing duplicate voting attempts. After successful vote submission, the system generates a live result display with candidate-wise vote counts, percentage analysis, winner highlighting, and animated result bars.

This project effectively demonstrates important frontend concepts such as form validation, DOM event handling, dynamic content generation, localStorage usage, result calculation, and responsive user interface design. With its election commission styled layout and secure digital workflow, it stands as a strong academic mini project for demonstrating awareness of online democratic technologies.

## Key Features
- Voter registration and eligibility validation
- Dynamic candidate selection interface
- One vote per user simulation using localStorage
- Duplicate vote prevention
- Vote submission confirmation page
- Automatic result calculation
- Winner announcement with percentage bars
- Responsive professional election UI

## Technologies Used
- HTML5
- CSS3
- Vanilla JavaScript
- Browser LocalStorage

## Purpose of the Project
This project was developed to simulate the concept of digital voting and to provide awareness on how secure online voting interfaces can be designed using frontend web technologies.
